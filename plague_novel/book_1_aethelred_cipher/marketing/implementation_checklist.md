# Self-Publishing Launch - Complete Checklist

## PHASE 1: FOUNDATION (Weeks 1-4)

### Website Setup
- [ ] Register domain name (randypellegrini.com or authorname.com)
  - Recommended: Namecheap, Google Domains ($12-15/year)
- [ ] Choose website platform:
  - **Option A:** WordPress (self-hosted, most flexible) - Bluehost $3-10/mo
  - **Option B:** Squarespace (easiest, all-in-one) - $16-23/mo
  - **Option C:** Wix (middle ground) - $16-27/mo
- [ ] Install theme (clean, readable, mobile-friendly)
- [ ] Create core pages:
  - [ ] Home page (use website_home_page.md)
  - [ ] About page (use website_about_page.md)
  - [ ] Books page (series info)
  - [ ] Blog page
  - [ ] Contact page
- [ ] Set up SSL certificate (security - usually free with hosting)
- [ ] Create email address (author@yourname.com)
- [ ] Add social media links
- [ ] Test on mobile/tablet/desktop

**Estimated cost:** $50-100 first year
**Time investment:** 8-12 hours

---

### Email List Setup
- [ ] Choose email service provider:
  - **Recommended:** ConvertKit ($29/mo up to 1,000 subscribers) - best for authors
  - **Budget:** MailerLite (Free up to 1,000 subscribers)
  - **Alternative:** Mailchimp (Free up to 500 subscribers)
- [ ] Create signup form
- [ ] Embed form on website (homepage, footer, about page)
- [ ] Create welcome sequence (use email_welcome_sequence.md)
- [ ] Set up automation triggers
- [ ] Create lead magnet (first 3 chapters as PDF)
- [ ] Design opt-in landing page
- [ ] Test signup flow end-to-end

**Estimated cost:** $0-29/mo
**Time investment:** 6-8 hours

---

### Social Media Setup
- [ ] Create X/Twitter account
  - [ ] Write bio (150 chars: "Author of THE AETHELRED CIPHER, historical thrillers about medieval conspiracies. Book 1 coming [DATE].")
  - [ ] Add profile photo
  - [ ] Add header image (book cover or medieval imagery)
  - [ ] Pin tweet with newsletter link
  - [ ] Follow 50 authors in your genre
  - [ ] Follow 50 readers/book bloggers
- [ ] Create Instagram account (optional but recommended)
  - [ ] Write bio with link to website
  - [ ] Create 9 grid posts before going public
  - [ ] Use consistent aesthetic/filters
- [ ] Create Goodreads Author account
  - [ ] Claim your author profile
  - [ ] Add bio and photo
  - [ ] Add book (even if not released yet)
  - [ ] Join reader groups in your genre
- [ ] Create content calendar (use social_media_strategy.md)
- [ ] Design Canva templates for posts
- [ ] Schedule first week of posts

**Estimated cost:** $0 (or $13/mo for Canva Pro)
**Time investment:** 10-12 hours initial, then 30-60 min/day

## PHASE 2: CONTENT CREATION (Weeks 5-8)

### Cover Design (EBOOK + PAPERBACK)
- [ ] Study comp titles (screenshot 15-20 covers you like)
- [ ] Decide budget ($600-1,200 for full print wrap)
- [ ] Choose platform (Reedsy, 99designs, Fiverr Pro)
- [ ] Shortlist 3-5 designers with print experience
- [ ] Send creative brief (use cover_design_brief.md)
- [ ] Review initial concepts
- [ ] Request revisions
- [ ] Approve final design
- [ ] Receive all file formats:
  - [ ] Ebook cover (2560x1600px JPG)
  - [ ] **Print cover with spine (6"x9" full wrap PDF)**
  - [ ] Social media sizes
  - [ ] Source files (PSD/AI)
- [ ] Calculate spine width (based on page count)
- [ ] Design back cover copy
- [ ] Test thumbnail visibility on Amazon

**Estimated cost:** $600-1,000 (recommended for print wrap)
**Time investment:** 6-8 hours + waiting time

---

### Book Description & Metadata
- [ ] Write book description (use book_description_amazon.md)
- [ ] Test description on beta readers
- [ ] Create 7-10 keyword variations
- [ ] Choose 3 Amazon categories (BISAC codes)
- [ ] Write author bio (150 words)
- [ ] Create author photo (professional headshot recommended)
- [ ] Draft press release
- [ ] Create one-sentence pitch
- [ ] Create elevator pitch (30 seconds)
- [ ] Create back cover copy (for paperback)

**Estimated cost:** $0 (or $200-500 for professional copywriter)
**Time investment:** 6-8 hours

---

### Sample Chapters & Lead Magnet
- [ ] Format first 3 chapters as standalone PDF
  - [ ] Include cover image
  - [ ] Add table of contents
  - [ ] Professional formatting (or use Atticus/Vellum)
  - [ ] Include "Read the rest" CTA with purchase link
- [ ] Create EPUB version (for Kindle app readers)
- [ ] Upload to website/email service
- [ ] Test download links
- [ ] Create landing page for sample
- [ ] Write social media posts promoting free chapters

**Estimated cost:** $0-100 (if hiring formatter)
**Time investment:** 3-4 hours

## PHASE 3: BOOK PRODUCTION (Weeks 9-12)

### Manuscript Formatting (EBOOK + PAPERBACK)
- [ ] Choose formatter:
  - **DIY Tools:** Atticus ($147 one-time - handles both ebook & print), Vellum ($250 Mac only)
  - **Professional:** Reedsy formatters ($300-600 for ebook + print)
- [ ] Format ebook (EPUB/MOBI)
  - [ ] Chapter headings
  - [ ] Page breaks
  - [ ] Table of contents
  - [ ] Front matter (title page, copyright, dedication)
  - [ ] Back matter (author note, preview of Book 2, CTA)
- [ ] **Format paperback (6"x9" trade paperback)**
  - [ ] Proper margins (0.5" inside, 0.375" outside)
  - [ ] Headers/footers with page numbers
  - [ ] Chapter starts on odd pages (right side)
  - [ ] Gutter spacing for binding
  - [ ] Professional interior layout
- [ ] Proofread formatted versions (both ebook and print)
- [ ] Test on multiple e-readers (Kindle, Kobo, Apple Books)

**Estimated cost:** $147-600
**Time investment:** 12-20 hours (DIY) or 3-4 hours (professional)

---

### ISBN & Distribution Setup (EBOOK + PAPERBACK)
- [ ] Decide on distribution:
  - **Wide:** Amazon + Apple + Kobo + B&N + Google (recommended)
  - **Exclusive:** Amazon KDP Select only (bonus: KU earnings, marketing tools)
- [ ] **Purchase ISBNs (REQUIRED for paperback)**
  - **US:** Bowker.com ($125 for 1, $295 for 10 - **GET 10**)
  - **Other countries:** National ISBN agency
  - You need 2 ISBNs minimum: one for ebook, one for paperback
  - **Don't use free Amazon ISBNs** - limits distribution options
- [ ] Set up accounts:
  - [ ] Amazon KDP (kdp.amazon.com) - ebook + paperback
  - [ ] **IngramSpark (ingramspark.com) - paperback distribution to bookstores/libraries ($49 setup)**
  - [ ] Draft2Digital (for wide ebook distribution)
  - [ ] OR individual accounts: Apple Books, Kobo, B&N, Google Play
- [ ] Set up payment/tax info on all platforms
- [ ] Create author central account (Amazon)

**Estimated cost:** $295-344 (ISBN 10-pack + IngramSpark setup)
**Time investment:** 6-8 hours

---

### Upload & Pre-Order Setup (EBOOK + PAPERBACK)
- [ ] **Amazon KDP ebook upload:**
  - [ ] Book details (title, author, description)
  - [ ] Upload manuscript file (EPUB/MOBI)
  - [ ] Upload ebook cover (2560x1600px)
  - [ ] Set pricing ($4.99-6.99 for 70K words)
  - [ ] Choose categories (3 max)
  - [ ] Add keywords (7 max)
  - [ ] Enable pre-order (10 days - 1 year out)
- [ ] **Amazon KDP paperback upload:**
  - [ ] Book details (same as ebook)
  - [ ] Upload print interior PDF
  - [ ] Upload print cover PDF (full wrap with spine)
  - [ ] Set pricing ($13.99-15.99 based on page count)
  - [ ] Choose paper type (white or cream)
  - [ ] Preview paperback proof online
- [ ] **IngramSpark paperback upload:**
  - [ ] Upload print interior PDF (same as KDP)
  - [ ] Upload print cover PDF
  - [ ] Set wholesale discount (40-55% for bookstore sales)
  - [ ] Enable returns (increases bookstore orders)
  - [ ] Set pricing (match Amazon)
- [ ] Wide ebook distribution (if applicable):
  - [ ] Upload to Draft2Digital
  - [ ] Mirror Amazon metadata
  - [ ] Distribute to Apple, Kobo, B&N, Google
- [ ] **Order proof copies:**
  - [ ] Order from Amazon KDP ($5-8 + shipping)
  - [ ] Order from IngramSpark ($3-5 + shipping)
  - [ ] Review physical proof carefully
  - [ ] Check margins, spine alignment, cover quality
  - [ ] Make corrections if needed
  - [ ] Re-order proof if major changes made
- [ ] Approve and publish all formats!

**Estimated cost:** $10-30 (proof copies)
**Time investment:** 6-10 hours across all platforms

### FORMAT COMPARISON:

**Ebook Only:** $650-1,400
- Limits you to 30% of historical thriller market
- No library sales
- No bookstore presence
- No signed copy giveaways

**Ebook + Paperback:** $1,400-3,350 (**RECOMMENDED**)
- Access to 95% of market (ebook + print buyers)
- Library sales possible
- Local bookstore consignment
- Author copies for marketing/events
- Professional credibility

**All Formats (ebook + paperback + hardcover + audiobook):** $7,000-12,000
- Only worth it if Book 1 proves successful
- Wait until you have sales data

## QUESTIONS? STUCK?

If you need help with any of these steps:
- Website setup issues → Check platform's help docs or YouTube tutorials
- Cover design unclear → Review comp titles on Amazon in your genre
- Email service confusion → ConvertKit has best author-specific tutorials
- Social media overwhelmed → Pick ONE platform, master it first
- Formatting problems → Atticus has extensive video tutorials
- Marketing lost → Focus on email list first, everything else secondary

**Remember:** Perfect is the enemy of done. Launch at B+ quality, improve with Book 2.
