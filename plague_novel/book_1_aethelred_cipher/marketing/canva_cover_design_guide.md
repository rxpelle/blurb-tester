# Canva Cover Design Guide: The Aethelred Cipher

## BEFORE YOU START: Important Considerations

**⚠️ CRITICAL WARNING:**
- Canva is **NOT recommended** for full print wrap covers (spine + back cover)
- Print wraps require precise spine calculations and bleed settings that Canva can't handle well
- **Recommendation:** Use Canva ONLY for ebook cover, hire professional for print wrap ($700-1,000)

**If you proceed with Canva anyway:**
- You can create the **ebook cover** (front only) in Canva
- For paperback, you'll need to hire a designer to adapt your Canva front cover into a full wrap
- This saves you ~$300-400 vs. full custom design

---

## PART 1: EBOOK COVER (Safe for Canva)

### Canvas Setup

**Step 1: Create Custom Size**
1. Open Canva Pro (required for commercial use licensing)
2. Click "Create a design"
3. Select "Custom size"
4. Enter dimensions:
   - **Width:** 1600 px
   - **Height:** 2560 px
   - **Orientation:** Portrait
5. Click "Create new design"

**Why these dimensions:**
- Amazon KDP standard for ebook covers
- 1.6:1 aspect ratio (industry standard)
- High enough resolution for quality display

---

### Color Palette (Concept 1: The Cipher)

Create these as saved colors in Canva:

**Primary Colors:**
- **Aged Parchment:** `#F4E8D0` (background)
- **Deep Blood Red:** `#8B1E1E` (cipher dots, danger accents)
- **Sapphire Blue:** `#003D82` (cipher dots, mystery)
- **Golden Yellow:** `#FFD700` (cipher dots, illumination)

**Secondary Colors:**
- **Ink Black:** `#1A1A2E` (title text, Latin fragments)
- **Antique Gold:** `#B8860B` (accents, aged details)
- **Weathered Brown:** `#5D4E37` (aging, stains)

**To save colors in Canva:**
1. Click any element
2. Click color square
3. Click "+" next to "Document colors"
4. Enter hex code
5. Repeat for all colors

---

### Background Layer

**Step 1: Create Parchment Texture**

**Option A: Use Canva Elements (Easiest)**
1. Click "Elements" in left sidebar
2. Search: "parchment texture"
3. Look for aged paper backgrounds
4. Drag to fill entire canvas
5. Adjust transparency to 80-90%
6. Change color to `#F4E8D0`

**Option B: Upload Your Own**
1. Find free parchment textures at:
   - Unsplash.com (search "old paper texture")
   - Pexels.com (search "parchment background")
2. Download high-res version (at least 2000px wide)
3. Upload to Canva
4. Drag to fill canvas
5. Adjust color overlay to `#F4E8D0`

**Step 2: Add Weathering Effects**
1. Elements → Search "grunge texture"
2. Add subtle stains/worn edges
3. Set blend mode to "Multiply"
4. Reduce opacity to 20-30%
5. Position at edges for aged look

### Middle Layer: Cipher Elements

**Step 1: Create Manuscript Page Look**

1. **Add Latin Text Fragment:**
   - Elements → Text
   - Search template: "medieval script" or "old manuscript"
   - OR manually create:
     - Font: "IM Fell DW Pica" (free in Canva)
     - Alternative: "Cinzel", "Trajan Pro", "Cormorant Garamond"
     - Size: 18-24px
     - Color: `#1A1A2E` (ink black)
     - Opacity: 60-70% (faded)
   - Type authentic Latin phrase (examples):
     - "Veritas numquam perit" (Truth never perishes)
     - "Quod scripsi, scripsi" (What I have written, I have written)
     - "In tenebris lux" (Light in darkness)
   - Position: Upper third or along margins

2. **Create Cipher Dots Pattern:**
   - Elements → Shapes → Circle
   - Create three circles:
     - **Red circle:** `#8B1E1E`, 40-50px diameter
     - **Blue circle:** `#003D82`, 40-50px diameter
     - **Yellow circle:** `#FFD700`, 40-50px diameter
   - Arrange in geometric pattern:
     - Triangle formation
     - Grid pattern (3x3 or 4x4)
     - Scattered along margins
   - Duplicate pattern 3-4 times across cover
   - Vary opacity (some 100%, some 70%)
   - Add subtle shadow for depth

3. **Add Key Symbol (Subtle):**
   - Elements → Search "antique key"
   - Find ornate medieval-style key
   - Color: `#5D4E37` (weathered brown)
   - Size: Small, 150-200px
   - Opacity: 30-40% (very faint watermark)
   - Position: Behind title or bottom third
   - Optional: Add "eye" detail in key handle
     - Elements → Search "eye icon"
     - Make very small, embed in key

4. **Add Europe Map (Faint Overlay):**
   - Elements → Search "medieval map Europe"
   - OR use simple outline map
   - Color: `#B8860B` (antique gold)
   - Opacity: 15-20% (barely visible)
   - Position: Background layer, slightly off-center
   - Optional: Add small dots at key locations:
     - Mainz, Strasbourg, Prague, Constantinople, York

### Top Layer: Typography

**Title: "The Aethelred Cipher"**

**Font Options (in order of preference):**
1. **Cinzel** (best for medieval + readable)
   - Weight: Bold or Black
   - Size: 110-140px
   - Letter spacing: -2 to 0

2. **Trajan Pro** (classic, strong)
   - All caps
   - Size: 100-130px

3. **IM Fell DW Pica** (authentic medieval feel)
   - Size: 120-150px
   - More decorative, less readable at small sizes

4. **Cormorant Garamond** (elegant serif)
   - Weight: Bold
   - Size: 110-140px

**Title Styling:**
- Color: `#1A1A2E` (ink black)
- Add stroke/outline:
  - Color: `#B8860B` (antique gold)
  - Width: 3-5px
- Add shadow:
  - Blur: 10
  - Offset: 5
  - Color: Black, 40% opacity
- Position: Upper half, centered
- Break into 2-3 lines if needed:
  - "THE AETHELRED"
  - "CIPHER"

**Subtitle: "Book 1: The Genesis Protocol"**

**Font:**
- Same family as title, lighter weight
- Size: 28-36px
- Color: `#5D4E37` (weathered brown)
- Position: Below title, 20-30px gap

**Author Name: "Randy Pellegrini"**

**Font:**
- Same family as title
- Size: 40-55px
- Color: `#1A1A2E` (ink black)
- Position: Bottom third
- Optional: Add small line above/below
  - Color: `#B8860B`
  - Width: 2px
  - Length: 200-300px

---

### Final Touches

**Step 1: Add Texture Overlays**
1. Elements → Search "paper texture overlay"
2. Add subtle grain/noise
3. Set to "Overlay" or "Multiply" blend mode
4. Opacity: 10-20%

**Step 2: Create Vignette (Darkened Edges)**
1. Elements → Shapes → Rectangle
2. Fill entire canvas
3. Color: Black
4. Set to "Multiply" blend mode
5. Add gradient: center (0% opacity) → edges (40% opacity)

**Step 3: Test Thumbnail Visibility**
1. Download cover at full size
2. Resize to 300px wide in image viewer
3. Can you read the title clearly?
4. If not: increase title size, add more contrast
5. Repeat until readable at thumbnail size

## PART 2: WHAT CANVA CAN'T DO (Hire Professional For This)

### Print Wrap Cover Requirements

**What you need for paperback:**
1. **Spine width calculation:**
   - Based on page count (280 pages) + paper type
   - Formula: (Pages ÷ Paper PPI) = Spine width
   - Cream paper: ~0.0025" per page = 0.7" spine
   - White paper: ~0.002" per page = 0.56" spine

2. **Full wrap dimensions for 6"x9" book:**
   - Front: 6" x 9"
   - Spine: 0.56"-0.7" x 9"
   - Back: 6" x 9"
   - Bleed: 0.125" on all edges
   - **Total width:** 12.125" to 12.825" (depending on spine)
   - **Total height:** 9.25"
   - **Total pixels at 300 DPI:** 3,638-3,848 x 2,775px

3. **Back cover elements:**
   - Book description (150-200 words)
   - Barcode space (2" x 1.2" bottom right)
   - Author bio/photo (optional)
   - Series info
   - Publisher info
   - Praise quotes (optional)

4. **Technical requirements:**
   - CMYK color (Canva uses RGB)
   - 300 DPI (Canva standard is 72-150 DPI)
   - Proper bleed and safe zones
   - Print-ready PDF format

**Why Canva can't handle this:**
- No spine width calculator
- RGB color space only (print needs CMYK)
- Limited precision for bleed/safe zones
- Can't export true 300 DPI for large formats
- Text positioning for spine is very difficult

## COMMON CANVA MISTAKES TO AVOID

❌ **Using free Canva fonts** - Many aren't licensed for commercial book covers
✅ **Stick to Google Fonts** - Cinzel, IM Fell, Cormorant are safe

❌ **Cluttered design** - Too many elements competing
✅ **Simple, focused** - Manuscript + cipher + strong title

❌ **Low contrast** - Title blends into background
✅ **Test at thumbnail size** - Must be readable at 300px wide

❌ **Ignoring safe zones** - Text too close to edges
✅ **Keep text 0.25" from all edges** - Allows for crop variation

❌ **Using Canva templates** - They look generic, everyone uses them
✅ **Build from scratch** - Following this guide creates unique cover

❌ **Skipping proof copy** - What looks good on screen ≠ print
✅ **Order test print** - Even for ebook-only, see how it looks physical
