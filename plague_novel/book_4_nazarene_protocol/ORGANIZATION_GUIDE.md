## 📝 NAMING CONVENTIONS

### Chapters:
- Format: `chapter_##_title.md`
- Example: `chapter_01_the_census.md`
- Two-digit numbering for proper sorting

### Planning Documents:
- Uppercase names: `BOOK_4_OUTLINE_jesus_era.md`
- Continuation plans: `CONTINUATION_PLAN_chapter_#.md`

### Archive Files:
- Session notes: `SESSION_CONTEXT_*.md`
- Old drafts: `*_DRAFT.md`

---

## 🔄 MAINTENANCE

### When Adding New Files:
1. **Chapters** → `manuscript/chapters/`
2. **Planning docs** → `planning/`
3. **Session notes** → `archive/session_notes/`
4. **Old drafts** → `archive/old_drafts/`
5. **Research** → `research/`

### Regular Cleanup:
- Move superseded drafts to `archive/old_drafts/`
- Archive completed session notes to `archive/session_notes/`
- Keep root directory clean (only README, PROGRESS_SUMMARY, and this guide)
