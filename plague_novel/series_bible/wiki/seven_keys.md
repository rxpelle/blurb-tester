# The Seven Bronze Keys

Created 1177 BCE by [Nefertari](characters/nefertari.md). Bronze (copper/tin alloy). Amplify genetic memory in carriers; useless for non-carriers. **Two incompatible taxonomies exist in the raw sources** — see [⚠️ taxonomy conflict](#️-taxonomy-conflict-system-a-vs-system-b) below.

- **Raw sources:** `../SERIES_BIBLE_master_timeline.md`, `../SERIES_BIBLE_seven_keys_tracker.md`, `../book_5_augustine_protocol/manuscript/chapters/` (chapters 1–9)

## ⚠️ Taxonomy conflict (System A vs System B)

The series has two competing key-naming systems and they don't reconcile cleanly. Flag for Randy to canonize one before Book 5 print.

### System A — `SERIES_BIBLE_master_timeline.md` + current wiki
**Defensive (4):** Living Key (a person), Pattern Eye, Memory Bridge, Distribution Network
**Offensive (3):** Blood Register, Catastrophe Clock, Culling Method

### System B — Book 5 manuscript (chapters 1–9, 45,411 words, drafted Jan 2026)
All seven are physical artifacts; defensive/offensive split not used in this taxonomy:
1. Pattern Eye
2. Bronze Mirror (Alexandria/Sketis)
3. Memory Stone (Jerusalem, Mount of Olives)
4. Speaking Scroll (Constantinople/Hagia Sophia)
5. Dream Tablet (Lérins Island)
6. Time Chain (location unknown — lost or hidden)
7. Truth Seal (location unknown — lost or hidden)

### Reconciliation possibilities
- **Pattern Eye** is the only name common to both systems — definitely the same artifact.
- **Memory Bridge** (System A) vs **Memory Stone** (System B): close but not identical. System A's journey (Mycenaean → Cyprus → Alexandria → Constantinople → Silk Road → Samarkand → Guangzhou → Sarah Chen) doesn't match System B's "Jerusalem baptistery foundation, guarded by Melania." Maybe two different artifacts — or Memory Bridge is Memory Stone after a later relocation.
- **Distribution Network** (System A — fragmentary, encoded across families and Thomas's 11-city pattern) doesn't map to any single physical artifact in System B.
- **The 3 offensive keys** (Blood Register, Catastrophe Clock, Culling Method) don't appear at all in the Book 5 manuscript — neither named nor referenced.

Below, defensive movement is given per System A (current canon), with **System B Book 5 detail** appended. **Decide which system is canonical before next ingest.**

## Continuity notes

- **⚠️ Iron Key vs. Bronze Key.** The bloodline tracker says [Wilhelm von Eltville](characters/wilhelm_von_eltville.md) "Possessed Iron Key (one of seven bronze keys)" and passed it to [Thomas](characters/thomas_of_eltville.md). Bronze (copper/tin alloy) ≠ iron — these are different materials.
- **Miriam recovers Pattern Eye in Byblos** during Book 4 (per bloodline tracker). Make sure the Pattern Eye location at start of Book 4 is Byblos, not "hidden in Phoenicia" generally.
- **Pattern Eye chain-of-custody contradiction — fully resolved 2026-04-24** by the Book 5 chapter 1 rewrite. The custody chain is now single-path: Helena → [Quintus Publius Scipio](characters/quintus_publius_scipio.md) (Jerusalem 327) → Monnica (Carthage 352) → Augustine (Ostia 387) → Possidius (Hippo 430). The older "410 CE Julia handoff" was deleted as a duplicate/contradiction and is not present in the current manuscript. The earlier "Marcus Publius's 4th-c descendant — name TBD" placeholder has been resolved to Quintus Publius Scipio.
- **City of God key-coordinate encoding** (System B): Augustine deliberately encodes the locations of Pattern Eye, Bronze Mirror, Memory Stone, Speaking Scroll, Dream Tablet, plus search patterns for Time Chain and Truth Seal, into specific books of *City of God* (XV–XXI). This makes *City of God* a treasure map for future carriers — and a structural reason it must be preserved through the Dark Ages.
