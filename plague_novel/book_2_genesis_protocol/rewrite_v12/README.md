## What This Rewrite Fixes

**Current Problem:**
- Voice is too clinical/intellectual (Crichton-style)
- Emotions are TOLD not FELT
- Too much technical exposition
- Characters feel distant
- "Pazzaz factor": 5/10

**Target:**
- Intimate, visceral voice (King-style)
- Readers FEEL the terror/grief/guilt
- Cut exposition by 30%
- Characters feel like real people
- "Pazzaz factor": 8-9/10

## How to Use This Structure

### 1. **PROMPTS Folder**
Contains 6 specialized prompts for voice transformation:
- `01_master_voice_transformation.md` - Use first, establishes voice
- `02_emotional_scene_intensifier.md` - For key emotional beats
- `03_dialogue_humanizer.md` - Make dialogue more real
- `04_opening_hook_transformer.md` - Hook readers immediately
- `05_exposition_cutter.md` - Cut info dumps
- `06_brutal_line_editor.md` - Final polish

### 2. **CHAPTERS Folder**
Individual chapter files for the rewrite:
- Work on one chapter at a time
- Each chapter is ~2000-4000 words (manageable chunks)
- Easier to track progress and iterate

### 3. **ORIGINAL_CHAPTERS Folder**
Backup of original text:
- Reference when rewriting
- Compare before/after
- Don't lose good parts

### 4. **Progress Tracking**
See `rewrite_progress.md` for:
- Which chapters are done
- Which need revision
- Notes on each chapter

### 5. **Style Guide**
See `style_guide.md` for voice consistency rules

## Rewrite Workflow

**For Each Chapter:**

1. **Read original** (from ORIGINAL_CHAPTERS)
2. **Use Master Voice Prompt** (PROMPTS/01) to rewrite
3. **Save rewritten version** to CHAPTERS folder
4. **Mark complete** in rewrite_progress.md
5. **Read aloud** - does it sound like someone TELLING you a story?
6. **If not visceral enough** - use Emotional Scene Intensifier (PROMPTS/02)
7. **Move to next chapter**

**Order to Tackle:**
1. Start with Prologue + Chapter 1 (get voice right)
2. Do all chapter openings next (hook readers)
3. Then emotional keystone scenes (Lagos, David's letter, etc.)
4. Then remaining chapters
5. Final polish with line editor

---

## Success Metrics

After rewriting first 3 chapters, test with readers:
- ❌ "That was interesting" = not working yet
- ✅ "Holy shit that hurt" = working
- ✅ "I couldn't stop reading" = working
- ✅ "I cried at the letter" = definitely working
