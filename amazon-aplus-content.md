## SETUP CHECKLIST

1. **Brand Story** — Set up once in Author Central → Brand → A+ Content. Appears on ALL listings.
2. **Book 1 A+ Content** — KDP → Marketing → A+ Content → Create for ASIN B0GJTGCHS2
3. **Book 2 A+ Content** — KDP → Marketing → A+ Content → Create for ASIN B0GPM973N1
4. **Book 3 A+ Content** — KDP → Marketing → A+ Content → Create for ASIN B0GSWLM1WV
5. **Images** — Generate with `cover_generate` / `cover_ebook` MCP tools (DALL-E + Pillow). Sizes: 970x600 headers, 300x300/300x400 thumbnails, 315x145 Brand Story. Or use cover-generator CLI: `cover-generator ebook "Title" --dalle`
6. **Review quotes** — Replace placeholder quotes in Book 2 Module 4 with actual verified Amazon review text
7. **Submit** — Each takes 24-48 hours for Amazon approval

**Tip:** KDP sometimes rejects A+ Content for "promotional language." If that happens, soften the CTAs and remove any pricing or discount mentions. The comparison charts almost always get approved on first try.
